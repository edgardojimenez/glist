#Project Description
Grocery list is a mobile web application.
POC to play with different technologies.

#Technologies
Client
* Web based Mobile
* Javascript
* jQuery Mobile, jQuery
* Knockout.js
* Labs.js (Loader)
* medium.js (Event emitter)
* modjs (Build)
* mocha.js (Test)

Backend (not included in this project <a href="https://github.com/edgardojimenez/mobws" >mobs</a>)
* C#
* Asp.net Web API
* Dapper
* SQL Server
* MS Test

#Features
* Single page app
* mobile web app
* Build process
* jQuery mobile UI
* CORS support
* Unit tests
* Hosted on Azure (websites) and AppHarbor
